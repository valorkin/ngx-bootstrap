6.19: Dropdowns Inside click example
====================================
**Primary Actor**: User

**Scope**: Ngx-bootstrap DEMO / BS version 3&4

**Goal**: Show user example with insideClick, which provide possibility to click inside dropdown without closing it

Main success scenario:
----------------------
1. User opens Dropdowns demo page
2. User clicks on Inside click sub-menu
3. User see clickable button "Button dropdown", be default dropdown is closed
4. When user clicks on it, then dropdown opened and there are 4 items with 1 separator
5. When user clicks on any item, then nothing happens, dropdown still opened
6. When user clicks on button again, dropdown closed

Variations:
-----------
2*. User scroll to Inside click sub-menu
