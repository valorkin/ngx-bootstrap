1.4: Alerts Dismissing example
=========================
**Primary Actor**: User

**Scope**: Ngx-bootstrap DEMO / BS version 3&4

**Goal**: Show user how to provide dismissing to alerts

Main success scenario:
----------------------
1. User open Alerts demo page
2. User click on Dismissing sub-menu
3. User see 3 alerts with types: success, info, danger
4. User able to click on each close button near each alert
5. User have 2 clickable buttons: "Toggle dismissible", "Reset"
6. When user click on close button, the alert disappeared
7. When user click "Reset" - then alerts back to default (3 alert with close buttons)
8. When user click "Toggle dismissible", then close buttons disappeared


Variations:
-----------
2*. User scroll to Dismissing sub-menu
